# License This!

Cross platform shell application that can add license information to the current directory with a simple:

Mac/Linux:
```
./license-this.sh
```

or 

Windows:
```
license-this.bat
```

## Instructions

To set your license information simply run the configuration wizard:

Mac/Linux:
```
./configure.sh
```

or Windows:
```
configure.bat
```
